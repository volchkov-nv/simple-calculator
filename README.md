- Kotlin

- MVVM
- Dagger 2
- Room
- ViewModel + LiveData
- Coroutines
- Custom View
- Recycler with custom Adapter
- Swipe
- Shared Preferences
- AAC
- Jetpack Navigation
- Timber
- Chain responsibility design pattern
- Facade design pattern
